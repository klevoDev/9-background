Бесшовные фоны

Бесшовные фоны очень удобно использовать, т.к. занимают очень мало места и сайт может их прогрузить намного быстрее, в отличии от изображений с высоким разрешением.

Попробуйте установить бесшовный фон на все секции с серым фоном.
background-repeat: repeat; прописывать не нужно, оно стоит по умолчанию

Изображения внутри блока

Попробуйте переместить изображние внутри блока с помощью background-position.

Списки

С помощью свойств background можно реализовать простые списки, но если текста будет больше, чем на одну строку, то будет не очень красиво.

  • Задний фон можно использовать для создания маркированных списков
  • Задний фон может расширяться засчёт css свойства padding
  • ВАЖНО: у последнего элемента списка необходимо убрать отсуп снизу!
  • Для стилизации можно использовать псевдоклассы!

Масштабирование

Данное изображение должно отображаться на всю ширину, независимо от разрешения экрана: как на 320px, так и на 1920px.
